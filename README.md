# 📇 GestionContacts

Application web de gestion de contacts développée avec Laravel dans le cadre d’un exercice académique de Master 1 GLAR.

Elle permet de gérer facilement des contacts avec une authentification sécurisée et une connexion via Google OAuth.

---

## 👨‍🎓 Contexte du projet

Ce projet a été réalisé par **Gnawé Parfait** dans le cadre d’un exercice de classe en Master 1 GLAR à Dakar.

L’objectif était de mettre en pratique :
- le développement web avec Laravel
- l’authentification sécurisée
- l’intégration d’une API externe (Google OAuth)
- la gestion CRUD complète

---

## 🚀 Fonctionnalités

- 🔐 Authentification classique (inscription / connexion)
- 🌐 Connexion via Google (Google OAuth)
- 👤 Gestion des utilisateurs
- 📇 CRUD complet des contacts (Créer, Lire, Modifier, Supprimer)
- 🔎 Recherche et organisation des contacts
- 📱 Interface moderne et responsive
- 🔒 Sécurisation des données utilisateurs

---

## 🔑 Authentification Google

L’application intègre l’API Google OAuth pour permettre une connexion rapide et sécurisée avec un compte Google.

---

## 🖥️ Aperçu

GestionContacts permet de :

- Créer un compte ou se connecter
- Se connecter via Google
- Ajouter et gérer ses contacts
- Accéder à ses données depuis n’importe quel appareil

---

## 🛠️ Technologies utilisées

- Laravel
- PHP
- MySQL
- Blade Templates
- Tailwind CSS
- Vite
- Google OAuth API
